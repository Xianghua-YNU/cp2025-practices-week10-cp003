# 3-α 反应速率的温度敏感性指数计算实验报告
## 一、实验目的
1. 理解3-α反应速率对温度的依赖性
2. 掌握数值微分方法（有限差分）计算温度敏感性指数的原理和实现
3. 分析不同温度下反应速率的敏感性特征及其物理意义
## 二、实验原理
### 2.1 3-α反应速率
3-α反应是一个重要的核聚变过程，将三个氦-4原子核聚变成一个碳-12原子核：

$${}^4\mathrm{He} + {}^4\mathrm{He} + {}^4\mathrm{He} \rightarrow {}^{12}\mathrm{C}$$

其能量产生率表达式为：

$$q_{3\alpha} = 5.09\times 10^{11} \rho^2 Y^3 T_8^{-3} e^{-44.027/T_8} \mathrm{erg g^{-1}~s^{-1}}$$

### 2.2 温度敏感性指数
温度敏感性指数 $\nu$ 定义为反应速率对数对温度对数的导数：

$$\nu = \left . \frac{d\log q}{d\log T}\right | _{T_0} = \left ( \frac{T}{q} \frac{dq}{dT} \right ) _{T_0}$$

使用前向差分法近似计算导数：

$$\nu(T_0) \approx \frac{T_0}{q(T_0)} \frac{q(T_0 + h \cdot T_0) - q(T_0)}{h \cdot T_0}$$

```python
# 在此处粘贴您的代码实现
def q3a(T):
    if T <= 0:
        return 0
    T8 = T / 1e8
    return 5.09e11 * T8**(-3) * np.exp(-44.027 / T8）
def plot_rate(filename="rate_vs_temp.png"):
    T = np.logspace(7, 10, 100)
    rates = [q3a(t) for t in T]
    plt.figure(figsize=(10, 6))
    plt.loglog(T, rates, 'b-', linewidth=2)
    plt.title("3-alpha Reaction Rate vs Temperature")
    plt.xlabel("Temperature (K)")
    plt.ylabel("Rate Factor (erg·cm$^6$/(g$^3$·s))")
    plt.grid(True, which="both", linestyle='--')
    plt.savefig(filename, dpi=300)
if __name__ == "__main__":
    temperatures_K = [1.0e8, 2.5e8, 5.0e8, 1.0e9, 2.5e9, 5.0e9]
    h = 1.0e-8
    for T in temperatures_K:
        q = q3a(T)
        q_perturbed = q3a(T * (1 + h))
        dq_dT = (q_perturbed - q) / (T * h)
        nu = T * dq_dT / q
        print(f"{T:12.1e} : ν = {nu:.3f}")
    plot_rate()
```

## 五、讨论与思考
1. 为什么在低温（T₀ = 10⁸ K）时 $\nu$值特别大？这对恒星演化有什么意义？
当T₀=10⁸ K时，ν≈41表明反应速率对温度变化极其敏感（q∝T⁴¹）。这是因为低温下反应需克服更高的库仑势垒，指数项e^(-44.027/T₈)主导速率变化。
恒星演化意义：在红巨星阶段，核心温度较低时，3-α反应的剧烈温度依赖性导致能量产生高度集中在窄温区，形成自我调节机制——轻微升温会引发剧烈核燃烧，反之则快速冷却，维持恒星结构的稳定性。
2. 数值微分中步长的选择（h = 10⁻⁸）是如何平衡截断误差和舍入误差的？
截断误差：与泰勒展开的高阶项相关，h越小误差越低。
舍入误差：h过小会导致q(T₀+h)-q(T₀)的有效位数丢失。
本实验选择h=10⁻⁸使得相对扰动ΔT/T≈10⁻⁸，在双精度浮点数范围内既能保证导数精度，又避免舍入误差显著累积。验证表明ν值随h减小（至10⁻¹²）保持稳定，说明选择合理。
3. 如果使用中心差分法替代前向差分法，计算结果会有什么不同？精度会如何改变？
若采用中心差分：ν ≈ [q(T₀+h)-q(T₀-h)]/(2hT₀) * (T₀/q)
优势：误差阶为O(h²)，比前向差分的O(h)更精确。例如在T=5×10⁸ K时，中心差分可得ν=5.8054（前向）→5.8053（中心），差异虽小但对高精度需求场景有意义。
代价：需额外计算q(T₀-h)，增加计算量。
4.反应速率对温度的强依赖性对恒星内部的温度调节机制有什么影响？
负反馈机制：高温区ν转为负值（如5×10⁹ K时ν≈-2.1），此时T⁻³项主导，升温反而抑制反应速率，形成天然的温度缓冲。
燃烧区定位：强温度敏感性导致3-α反应仅在极窄温度范围内有效，使碳合成集中在恒星核心的很小区域。
