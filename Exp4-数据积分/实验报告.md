## 二、实验原理

### 2.1 梯形积分法

对于离散的速度数据点$(t_0,v_0),(t_1,v_1),...,(t_N,v_N)$，总距离（位移）可通过梯形法则计算：

$$
\text{距离} \approx \sum_{i=0}^{N-1} \frac{v_i + v_{i+1}}{2} (t_{i+1} - t_i)
$$

### 2.2 累积距离计算

使用`scipy.integrate.cumulative_trapezoid`计算每个时间点的累积距离：

$$
\text{累积距离}(t_i) = \int_{t_0}^{t_i} v(t)dt
$$

## 三、实验代码

请粘贴你的完整Python实现代码：

```python
import numpy as np
from scipy.integrate import cumulative_trapezoid
import matplotlib.pyplot as plt

def main():
    try:
        # 数据文件路径
        data_file = r'E:\计算物理\Velocities.txt'

        # 读取数据
        data = np.loadtxt(data_file)
        t = data[:, 0]  # 时间列
        v = data[:, 1]  # 速度列

        # 计算总距离
        total_distance = np.trapz(v, t)
        print(f"总运行距离: {total_distance:.2f} 米")

        # 计算累积距离
        distance = cumulative_trapezoid(v, t, initial=0)

        # 绘制图表
        plt.figure(figsize=(10, 6))

        # 绘制速度曲线
        plt.plot(t, v, 'b-', label='Velocity (m/s)')

        # 绘制距离曲线
        plt.plot(t, distance, 'r--', label='Distance (m)')

        # 图表装饰
        plt.title('Velocity and Distance vs Time')
        plt.xlabel('Time (s)')
        plt.ylabel('Velocity (m/s) / Distance (m)')
        plt.legend()
        plt.grid(True)

        # 显示图表
        plt.show()
    except FileNotFoundError:
        print(f"错误：找不到数据文件 {data_file}")
        print("请确保数据文件存在于指定目录中")

if __name__ == '__main__':
    main()
    

```

### 4.2 误差分析
1.梯形积分法是基于将积分区间划分为若干个小区间，然后用梯形面积之和来近似积分值。它假设在每个小区间内，被积函数（速度函数）是线性变化的。然而，实际的速度函数可能并非严格线性，这种对函数的线性近似会导致截断误差。
2.在实际应用中，速度 - 时间数据通常是离散的，即我们只能获取到有限个时间点上的速度值。梯形积分法是基于这些离散的数据点进行计算的，而实际的速度函数是连续的。这种从连续函数到离散数据的转换会引入误差。
